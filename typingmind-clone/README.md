# TypingMind Clone

یک رابط کاربری چت پیشرفته برای مدل‌های هوش مصنوعی مشابه TypingMind

## ویژگی‌ها

- ✅ پشتیبانی از چندین مدل AI:
  - OpenAI (GPT-4, GPT-3.5)
  - Anthropic (Claude)
  - Google (Gemini)
  - Ollama (مدل‌های محلی)
  
- ✅ مدیریت مکالمات (ذخیره، جستجو، دسته‌بندی)
- ✅ کتابخانه Prompt
- ✅ حالت تاریک/روشن
- ✅ واکنش‌گرا (Responsive)
- ✅ ذخیره‌سازی محلی با IndexedDB
- ✅ استریمینگ پاسخ در لحظه
- ✅ امنیت API Keys (ذخیره محلی)

## تکنولوژی‌ها

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: TailwindCSS
- **State Management**: Zustand
- **Storage**: IndexedDB (idb)
- **Markdown**: react-markdown
- **Icons**: lucide-react

## نصب و راه‌اندازی

### پیش‌نیازها

- Node.js 18+ 
- npm یا yarn

### مراحل نصب

1. نصب وابستگی‌ها:
```bash
cd typingmind-clone
npm install
```

2. اجرای سرور توسعه:
```bash
npm run dev
```

3. باز کردن مرورگر در آدرس:
```
http://localhost:3000
```

## نحوه استفاده

1. **تنظیم کلیدهای API**:
   - روی دکمه تنظیمات (⚙️) کلیک کنید
   - کلید API ارائه‌دهنده مورد نظر خود را وارد کنید
   - برای Ollama می‌توانید URL سرور محلی را تنظیم کنید

2. **شروع گفتگو**:
   - روی دکمه "+" کلیک کنید تا گفتگوی جدید ایجاد شود
   - مدل و ارائه‌دهنده مورد نظر را انتخاب کنید
   - پیام خود را تایپ کرده و ارسال کنید

3. **کتابخانه Prompt**:
   - روی دکمه کتاب (📖) کلیک کنید
   - Promptهای جدید ایجاد کنید
   - با کلیک روی هر Prompt، محتوای آن در ورودی چت قرار می‌گیرد

4. **مدیریت گفتگوها**:
   - گفتگوها به صورت خودکار ذخیره می‌شوند
   - از نوار کناری می‌توانید بین گفتگوها جابجا شوید
   - با کلیک روی آیکون سطل زباله می‌توانید گفتگو را حذف کنید

## افزودن مدل‌های جدید

برای افزودن پشتیبانی از یک ارائه‌دهنده جدید:

1. تابع مربوطه را به `src/lib/ai/ai-service.ts` اضافه کنید
2. نوع provider را در `src/types/index.ts` به‌روز کنید
3. فیلد API Key مربوطه را در Settings اضافه کنید

## امنیت

- تمام کلیدهای API به صورت محلی در مرورگر ذخیره می‌شوند
- هیچ داده‌ای به سرورهای شخص ثالث ارسال نمی‌شود (به جز درخواست‌های مستقیم به APIهای AI)
- داده‌های چت در IndexedDB مرورگر ذخیره می‌شوند
